# 5-3-1 API

REST API for

Run `npm install` and then `npm run ...`:

| Command     | Description                 |
| ----------- | --------------------------- |
| start       | Start Node                  |
| start:debug | Start Node w/debugger       |
| build       | Build Typescript to ./build |
| server      | Start Server                |
| test        | Run tests                   |
| test:cove   | Run tests w/coverage        |
| test:debug  | Run tests w/debugger        |
